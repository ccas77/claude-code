Three components, one pipeline

All state lives in a local database, so a run can crash — or you can walk away for a month — and the next run resumes cleanly.

1 · Image generator

1000×1500 pins in five templates, per-pen-name color palettes, bundled open fonts, and deterministic seeds so re-runs don't spam near-duplicates. Backgrounds are generated locally — nothing pulled from the web.

2 · Copy generator + review

Keyword-first titles and reader-search descriptions written for how readers actually browse, drafted per variant and approved by you in a local gallery. Nothing publishes unapproved.

3 · Boards + scheduler

Themed boards per pen name and a publishing scheduler that runs for months on a cron, spacing pins so the account never looks spammy.

  • Anti-spam by design: a weekly cap, the same image never published twice, ≥48 hours between publishes of the same link, round-robin across brands, one re-save after five days, and automatic back-off & quarantine on errors.
  • No fabricated data: it asks you or leaves fields blank — it never invents titles, links, tropes, or stats.
