# Pitch Materials

Editor- and agent-ready sell assets for *The Friend Who Never Says No*. All claims match the canonical facts in the top-level `README.md`; full evidence lives in `../04-market-evidence.md`.

| File | What it is |
|---|---|
| `one-page-pitch.html` | A visual one-page pitch sheet (self-contained; open in any browser or print to PDF) |
| `sell-sheet.md` | The one-page sell sheet in text form (title, logline, hook, why-now, audience, differentiators, risk) |
| `back-cover-copy.md` | Back-cover copy, flap copy, and a one-line catalog description (three lengths) |
| `title-and-positioning-workshop.md` | Title/subtitle alternatives with trade-offs, taglines, the sales-conference positioning line, franchise naming |
| `agent-query-letter.md` | A model agent query / pitch letter an attached author can adapt |

The formal book proposal is `../05-book-proposal.md`; the differentiating Part III sample is `../07-part-III-playbook-sample.md`.
